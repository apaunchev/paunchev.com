---
title: 'Rename a commit'
updatedAt: '2022-01-02 12:00:00'
---

To rename the most recent commit:

```bash
git commit --amend -m "The correct message"
```

If the commit is older, then do an interactive rebase:

```bash
git rebase -i [hash]
```

In the editor, change `pick` to `reword` and save.
