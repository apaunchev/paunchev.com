---
title: 'Join classes together'
updatedAt: '2022-01-02 12:00:00'
---

A simplified way to do (some of) what the [classnames](https://github.com/JedWatson/classnames) package does.

```javascript
const classNames = (...args) => args.filter(Boolean).join(' ');
```

Usage:

```javascript
classNames('page-header', isCentered && 'page-header--centered');
```
