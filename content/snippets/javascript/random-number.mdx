```javascript
const random = (min, max) => Math.floor(Math.random() * (max - min)) + min;
```

Usage:

```javascript
// Get a random number out of [10, 11, 12, 13]
random(10, 14);
// Get a random number from 1 to 100 (inclusive)
random(1, 101);
// Get a random number from -10 to 10 (inclusive)
random(-10, 11);
```
