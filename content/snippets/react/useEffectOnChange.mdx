---
title: 'useEffectOnChange'
updatedAt: '2022-03-26 13:00:00'
---

A thin wrapper around “useEffect” which will only fire when the value changes, and not on mount.

```jsx
function useEffectOnChange(callback, deps) {
  const hasMounted = React.useRef(false);

  React.useEffect(() => {
    if (!hasMounted.current) {
      hasMounted.current = true;
      return;
    }

    callback();
  }, deps);
}
```
